# Weatherinformation-cli
...
a simple CLI used to check the current weather at your location based on IP Geolocation...

## Usage:

```
npm install -g weatherinformation-cli

weather
```

```

---

Humidity:           72.4

Pressure:           998.4 hPa
---
```
